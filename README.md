# Credit_Card_Fraud_Detection
AI-powered Credit Card Fraud Detection System built with Python, Machine Learning and Streamlit.

# 🛡️ FraudShield AI

## 🚀 Features
- 💳 Credit Card Fraud Detection
- 📊 Interactive Dashboard
- 📈 Analytics Charts
- 🤖 AI-based Prediction
- ⚡ Fast & User-Friendly Interface
- 📥 Download Report

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
